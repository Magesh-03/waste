      What is JavaScript?

    1. JavaScript is a high-level, interpreted programming language primarily used for client-side web development. It allows you to create dynamic and interactive content on web pages. History of JavaScript: JavaScript was developed by Brendan Eich at Netscape Communications Corporation in 1995. Initially named LiveScript, it was later renamed JavaScript when it was standardized as ECMAScript by ECMA International.

      2. Features of JavaScript:

    2. Client-Side Scripting: JavaScript is mainly used for client-side scripting, meaning it runs on the user's browser and can manipulate the Document Object Model (DOM) of a web page. Dynamic and Interactive: JavaScript allows you to create dynamic and interactive web elements, such as animations, form validations, and interactive menus. Event Handling: You can handle various events like mouse clicks, keyboard input, form submissions, etc., using JavaScript. Asynchronous Programming: JavaScript supports asynchronous programming using callbacks, promises, and async/await, which is essential for handling tasks like fetching data from servers without blocking the UI. Cross-Platform: JavaScript is supported by all major web browsers, making it a cross-platform language for web development. Where JavaScript is Used: JavaScript is primarily used for:

    3. let, const, var Data Types: Number, String, Boolean, Array, Object, Null, Undefined Operators: Arithmetic, Comparison, Logical, Assignment, Ternary Control Structures: if...else, switch, for loop, while loop, do...while Functions: Declaring functions, Arrow functions, Callbacks, Higher-order functions Objects and Classes: Creating objects, Prototypes, Classes (ES6+) DOM Manipulation: Accessing and modifying HTML elements, Event handling

      4. Libraries and Frameworks:

      JavaScript has a vast ecosystem of libraries and frameworks that extend its capabilities and simplify development tasks. Some popular libraries/frameworks include jQuery, React.js, Angular, Vue.js, Express.js, and many more. .

    JavaScript-for-loop

    1. The for loop in JavaScript is a control flow statement that allows you to execute a block of code repeatedly based on a specified condition. It's commonly used when you know the number of iterations in advance. Here's the syntax of a for loop:
      2. for (initialization; condition; increment/decrement) { // Code to be executed }
      • Initialization: It's where you initialize the loop variable. This is executed only once before the loop starts.
      • Condition: It's the condition that is checked before each iteration. If the condition evaluates to true, the loop continues. If false, the loop terminates.
      • Increment/Decrement: It's used to update the loop variable after each iteration. It can be an increment (i++) or decrement (i--) operation.
      • Here's an example of a for loop that counts from 1 to 5:
        • for (let i = 1; i <= 5; i++) { console.log(i); // Outputs: 1, 2, 3, 4, 5 }

    javascript-Cookes

      Setting a cooke

    1. To set a cookie, you use the document.cookie property. The cookie value is a string in the format name=value; expires=expiration_date; path=path_value
      2. document.cookie = "username=John Doe; expires=Thu, 31 Dec 2024 23:59:59 GMT; path=/";

      Getting a Cokkies

    2. To retrieve a cookie, you can access the document.cookie property. It returns a string containing all cookies for the current domain.
      3. let allCookies = document.cookie; console.log(allCookies); // Outputs: "username=John Doe; other_cookie=value;"

      Cookie Options:

    3. Secure: You can set the secure attribute to ensure that the cookie is sent only over HTTPS connections. SameSite: This attribute helps prevent cross-site request forgery (CSRF) attacks by restricting when cookies are sent. Domain: Specifies the domain for which the cookie is valid. By default, it's valid for the current domain.
      4. document.cookie = "username=John Doe; secure; SameSite=Strict; domain=example.com; path=/";

    JavaScript-loop Control

    1. break statement:

      • The break statement is used to exit a loop prematurely, without completing all iterations.
      • It's commonly used to terminate a loop based on certain conditions
        • for (let i = 0; i < 5; i++) { if (i === 3) { break; // Exit the loop when i equals 3 } console.log(i); }

    2. continue statement:

      • The continue statement is used to skip the current iteration of a loop and proceed to the next iteration. It's often used to skip specific values or conditions within a loop.
        • for (let i = 0; i < 5; i++) { if (i === 2) { continue; // Skip the current iteration when i equals 2 } console.log(i); }

    3. return statement in functions:

      • In JavaScript functions, the return statement is used to exit the function and optionally return a value.
      • When used inside a loop in a function, it can terminate the loop and exit the function early.
        • function findNumber(arr, num) { for (let i = 0; i < arr.length; i++) { if (arr[i] === num) { return true; // Exit the loop and function if the number is found } } return false; // Return false if the number is not found } const numbers = [1, 2, 3, 4, 5]; console.log(findNumber(numbers, 3)); // Output: true console.log(findNumber(numbers, 6)); // Output: false
